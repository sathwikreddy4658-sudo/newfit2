## Toolbar Buttons Explained

### Text Formatting Buttons
- **B** (Bold) - Makes text bold and darker
- **I** (Italic) - Slants text for emphasis
- **U** (Underline) - Adds line under text
- **S** (Strikethrough) - Draws line through text

### Paragraph Formatting
- **""** (Blockquote) - Creates highlighted quote block with left border
- **<>** (Code Block) - Creates monospace code area

### Heading Buttons
- **H1** - Large main heading
- **H2** - Smaller subheading

### List Buttons
- **1.** (Ordered List) - Numbered list (1, 2, 3...)
- **‚Ä¢** (Bullet List) - Bulleted list (dots)

### Other Buttons
- **üîó** (Link) - Add hyperlink to text
- **X** (Clear) - Remove all formatting from selection

## How to Use Each Feature

### 1. Make Text Bold
1. Select the text you want bold
2. Click the **B** button in toolbar, OR
3. Press `Ctrl + B` (Windows/Linux) or `Cmd + B` (Mac)
4. Text will become darker and thicker

### 2. Make Text Italic
1. Select the text you want italic
2. Click the **I** button in toolbar, OR
3. Press `Ctrl + I` (Windows/Linux) or `Cmd + I` (Mac)
4. Text will slant to the right

### 3. Underline Text
1. Select the text you want underlined
2. Click the **U** button in toolbar, OR
3. Press `Ctrl + U` (Windows/Linux) or `Cmd + U` (Mac)
4. A line will appear under the text

### 4. Create a Heading
1. Click where you want the heading
2. Click **H1** for main heading or **H2** for subheading
3. Type your heading text
4. The text will be larger and use heading font

### 5. Create a List
1. Click where you want the list
2. Click **1.** for numbered list or **‚Ä¢** for bullet list
3. Type your first item and press Enter
4. Each new line becomes a new list item
5. Press Enter twice to exit the list

### 6. Add a Quote/Blockquote
1. Click where you want the quote
2. Click the **""** (Blockquote) button
3. Type your quote text
4. Text will appear in a highlighted box with left border
5. Click the button again or press Backspace twice to exit

### 7. Add Code Block
1. Click where you want code
2. Click the **<>** (Code Block) button
3. Type your code
4. Text will use monospace font (good for code)
5. Click the button again to exit

### 8. Add a Link
1. Type or select the text you want to be a link
2. Click the **üîó** (Link) button
3. Paste the URL in the dialog box
4. Click OK or press Enter
5. The text will become blue and clickable

### 9. Remove Formatting
1. Select the text with formatting you want to remove
2. Click the **X** (Clear) button
3. All formatting will be removed
4. Text will return to normal

### 10. Undo/Redo
- Press `Ctrl + Z` (Windows/Linux) or `Cmd + Z` (Mac) to undo last change
- Press `Ctrl + Shift + Z` (Windows/Linux) or `Cmd + Shift + Z` (Mac) to redo
- You can undo multiple times to go back several steps

## Tips & Tricks

1. **Select Text Quickly**: Click at the start, then Shift+Click at the end to select
2. **Copy Formatting**: Select formatted text, then select plain text and apply formatting
3. **Multi-line Selection**: Click and drag to select multiple lines
4. **Delete Formatting**: Use the Clear button (X) to remove all formatting at once
5. **Undo Multiple Times**: Keep pressing Ctrl+Z to undo several changes
6. **Lists**: Press Tab to indent list items, Shift+Tab to outdent

## Common Issues & Solutions

### Issue: Formatting not showing in the blog
**Solution**: Make sure you clicked the Save button to save the blog. Formatting won't appear if blog isn't saved.

### Issue: Can't undo something
**Solution**: The undo history only goes back so far. If you need to recover, try refreshing the page (your draft is saved).

### Issue: Text looks too small/large
**Solution**: This is the default size. The blog will display at a readable size on the website.

### Issue: Link not working
**Solution**: Make sure the URL starts with `http://` or `https://`
